# CursiveRec
Python GUI Application that recognizes cursive writing. Recognizes lower-case cursive alphabets and 
numbers using single-stroke gestures. An alphabet is already provided (defaultGestures.pkl; refer to Guide.png),
but the user can also record their own set of gestures for later.

Uses the approach explained [here](http://depts.washington.edu/acelab/proj/dollar/index.html).
